If you have any comments, concerns, or any questions in general, feel free to reach out any time at (812)-512-1644, or my email at codyrg0228@gmail.com

What is running this site?

EquiSync is being hosted on an AWS (Amazon Web Service) EC2 instance. Including a discrete PostgreSQL database, an Nginx reverse proxy routing traffic to a Gunicorn WSGI daemon, and Cloudflare DNS managing secure SSL encryption protocols. I've learned quite a bit of information making this site alone, and I hope to continue this trend.

Python / Django PostgreSQL AWS EC2 Nginx Gunicorn Cloudflare
